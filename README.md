# Javascript/Typescript Reusable Utilities

This is a collection of reusable utilities for Javascript and Typescript functionalities.

## Prerequisites

- [Bun.js](https://bun.sh/) installed

### Installing dependencies

```bash
bun install
```

## Testing

### Running Tests

```bash
bun test # Run all tests
bun test --watch # Run tests in watch mode
bun test --coverage # Generate test coverage report
bun test --coverage --watch # Generate test coverage report in watch mode
```
